# Data_Structures_and_Algorithms
A simple library of the basic but fundamental (think like the STL but not as polished and optimized) data structures written in C++
Some of the data structures included:
  - Stacks (done)
    - Operations:
      - get_top
      - push
      - pop
  - Queues (done)
    - Operations:
      - get_front
      - get_back
      - enqueue
      - dequeue
  - Linked Lists (done)
    - Operations
      - get_head
      - add_back
      - add_front
      - search
      - remove_back
      - remove_front
      - update_value
      - update_key
  _ Hash Tables
  - Heaps
  - Trees
